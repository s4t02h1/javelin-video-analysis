# β版フィードバックフォーム仕様書

このドキュメントは、β版利用後のフィードバックフォームの設計仕様です。
Google フォームまたは同等のツールで作成し、ダッシュボードにリンクを表示します。

---

## フォームタイトル

やり投げ動画解析 β版 フィードバックフォーム

## フォーム説明文

```
β版をご利用いただきありがとうございます。
ご感想・気になった点・不具合などをお聞かせください。
所要時間：3〜5分程度
いただいたフィードバックはサービス改善に活用させていただきます。
```

## API フィードバック送信仕様（将来対応）

フォーム送信データを将来的に Feedback API に送信する場合のマッピング:

```
POST /v1/public/feedback
{
  "dashboard_token": "<回答15から抽出>",
  "feedback_type": "<不具合があれば'bug', なければ'report_quality' 等>",
  "severity": "<重大度を判定>",
  "title": "<自動生成またはフォーム回答から>",
  "body": "<回答6 + 回答8 + 回答12 を結合>",
  "device": "<回答9>",
  "os": "<回答9から推定>",
  "browser": "<回答10>"
}
```

## フィードバック集計方法

1. Google フォームのスプレッドシートを確認する
2. 管理画面「フィードバック管理」タブから手動で入力する（または API 連携）
3. 重大な不具合は severity=critical で登録し、改善ログに追加する
4. 「β版KPI」タブでフィードバック提出率を確認する

---

> **注意**: フィードバックフォームへのリンクは `configs/beta_release_config.yaml` の `feedback_form_url` に設定してください。ダッシュボードのフィードバックバナーにリンクが表示されます。
