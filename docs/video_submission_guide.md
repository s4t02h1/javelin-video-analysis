# 動画提出ガイド

本ページではやり投げ動画解析サービスへの動画提出方法をご案内します。

---

## 1. 動画を撮影する前に

| 確認事項 | 推奨 |
|---|---|
| カメラの位置 | 腰〜肩の高さ、横（側面）か斜め後方 |
| フレームレート | 60fps 以上（スローモーション推奨） |
| 解像度 | 1280×720（HD）以上 |
| 背景 | 無地・明るい場所（芝、空など） |
| 服装 | 体の輪郭が見えるフィット感のある服 |

---

## 2. 推奨する撮影アングル

### 側面（横から）
- 最も解析精度が高いアングルです。
- カメラを助走方向に対して真横に配置してください。
- 投てき全体が画角に入るようにしてください。

### 斜め後方
- 踏み込み・クロスステップが確認しやすいアングルです。
- 投げる方向に対して45〜60度後方から撮影してください。

---

## 3. 撮影方法

1. カメラをスタンドや三脚に固定する（手ブレを防ぐ）
2. 被写体（選手）が全身フレーム内に収まることを確認する
3. スローモーション（240fps / 120fps）推奨
4. 1投ごとにファイルを区切る（1投 = 1ファイル）
5. ファイル名に日付・試技番号を入れる（例: `20260510_01.mp4`）

---

## 4. 動画ファイルの形式

| 項目 | 対応形式 |
|---|---|
| 拡張子 | `.mp4` `.mov` `.avi` `.mkv` |
| 最大サイズ | 2GB まで（管理者へ直接送付の場合は制限なし） |
| 最小長さ | 助走開始〜投げ後の着地（フォロースルー含む）まで |

---

## 5. 提出方法

管理者から指定された方法でファイルを送付してください。

- **クラウドストレージ共有**（Google Drive / Dropbox など）
- **SNS の DM 添付**（ファイルサイズ制限に注意）
- **直接メッセージ送付**（LINE / Instagram DM など）

---

## 7. よくある質問

**Q: スマートフォンで撮影しても大丈夫ですか？**  
A: はい。iPhone / Android どちらでも対応しています。スローモーションモードの利用を推奨します。

**Q: 複数本送っても大丈夫ですか？**  
A: はい。優先的に解析したい動画の番号やコメントをあわせてお伝えください。

**Q: 動画が鮮明でなくても解析できますか？**  
A: 解析は可能ですが、画質・撮影角度・背景の条件によって精度が変わります。あらかじめご了承ください。

**Q: いつ納品されますか？**  
A: 受付後、状況により数日〜1週間程度いただく場合があります。

## 8. 同意事項

本サービスをご利用いただくにあたり、下記の各項目にご同意ください。

1. **参考資料である旨の同意**: 本解析は参考情報であり、絶対評価・診断結果ではありません。
2. **医療診断ではない旨の同意**: 怪我の診断・医療行為には当たりません。
3. **競技指導の代替ではない旨の同意**: 専門コーチや競技指導の代替にはなりません。
4. **解析精度の変動**: 動画の画質・角度・服装・背景により解析精度が変わる場合があります。
5. **納品時間**: 解析・納品まで時間がかかる場合があります。
6. **SNS掲載は別途許可制**: 解析結果のSNS掲載を希望する場合は別途ご連絡ください。

ご不明な点があれば管理者にお気軽にお問い合わせください。
