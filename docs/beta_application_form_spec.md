# β版申込フォーム仕様書

このドキュメントは、β版利用申込フォーム（Google フォームまたは Web フォーム）の設計仕様です。
収集した情報は intake API または管理画面の「受付一覧」に入力することを想定しています。

---

## フォームタイトル

やり投げ動画解析 β版 申込フォーム

## フォーム説明文

```
やり投げ動画解析β版にご興味いただきありがとうございます。
以下の項目にご回答ください。β版は先着 N 名限定です。

【ご注意】
・β版のため、解析内容や表示に不完全な点が残る場合があります。
・解析結果は参考資料です。医療診断・専門的競技指導を代替するものではありません。
・未成年の方がご利用の場合は、保護者または指導者の確認をおすすめします。
```

## フォーム項目一覧

### セクション1: 基本情報

| # | 項目名 | 種別 | 必須 | 対応フィールド | 備考 |
|---|--------|------|------|----------------|------|
| 1 | お名前 / ニックネーム | テキスト (短) | ✅ | `name_or_nickname` | 本名不要 |
| 2 | 連絡先（LINE または Instagram） | テキスト (短) | ✅ | `contact` | 納品に使用する連絡手段 |
| 3 | Instagram アカウント名（任意） | テキスト (短) | — | `instagram_account` | @ マークなしで入力 |

### セクション2: 競技情報

| # | 項目名 | 種別 | 必須 | 対応フィールド | 選択肢 |
|---|--------|------|------|----------------|--------|
| 4 | カテゴリ | ラジオボタン | ✅ | `age_group` | 高校生 / 大学生 / 社会人・マスターズ / 指導者 / その他 |
| 5 | 利き腕 | ラジオボタン | ✅ | `dominant_arm` | 右 / 左 / 両方 |
| 6 | 競技歴（おおよそ） | ラジオボタン | — | `experience_years` | 1年未満 / 1〜3年 / 3〜5年 / 5年以上 |
| 7 | 自己ベスト（メートル）（任意） | テキスト (短) | — | `personal_best` | 例: 42.50m |

### セクション3: 動画について

| # | 項目名 | 種別 | 必須 | 対応フィールド | 選択肢 / 備考 |
|---|--------|------|------|----------------|---------------|
| 8 | 撮影角度（予定） | ラジオボタン | ✅ | `shooting_angle` | 真横（側面）/ 斜め後方 / 正面 / その他 |
| 9 | スロー撮影 | ラジオボタン | — | `is_slow_motion` | はい（スロー） / いいえ（通常速度） |
| 10 | 動画の本数 | ラジオボタン | ✅ | `video_count` | 1本 / 2本（成功と失敗など） |

### セクション4: ご希望・ご相談

| # | 項目名 | 種別 | 必須 | 対応フィールド | 備考 |
|---|--------|------|------|----------------|------|
| 11 | 解析に期待すること | テキスト (長) | — | `main_request` | 例: リリース角度を確認したい |
| 12 | 気になっている点 | テキスト (長) | — | `focus_points` | 自由記述 |

### セクション5: 同意事項

以下は全て必須チェックボックス（未チェックでは送信できない）。

| # | ラベル | 対応フィールド |
|---|--------|----------------|
| 13 | 解析結果は参考資料であることに同意する | `consent_reference_analysis` |
| 14 | 医療診断・怪我の診断を代替するものではないことに同意する | `consent_not_medical` |
| 15 | 専門的な競技指導の代替ではないことに同意する | `consent_not_coaching_replacement` |
| 16 | 動画の画質・撮影角度・服装等により精度が変わることに同意する | `consent_accuracy_depends_on_video` |
| 17 | 納品まで数日〜1週間程度かかる場合があることに同意する | `consent_delivery_may_take_time` |
| 18 | SNS掲載は別途確認すること（申込 = SNS掲載許可ではない）に同意する | `consent_sns_requires_permission` |

### セクション6: SNS・教師データ（任意）

| # | 項目名 | 種別 | 必須 | 対応フィールド | 選択肢 |
|---|--------|------|------|----------------|--------|
| 19 | SNS（Instagram等）への動画・画像掲載 | ラジオボタン | — | `sns_permission_status` | 許可する / 匿名（名前なし）なら許可 / 不可 / まだ決めていない |
| 20 | AI学習データ（教師データ）への利用 ※SNS掲載許可とは別 | ラジオボタン | — | `training_data_consent` | 同意する / 同意しない / まだ決めていない |

## intake API マッピング

フォーム回答を受け取ったら、以下の手順で intake を作成する。

```
POST /v1/intake
{
  "source": "google_form",
  "name_or_nickname": "<回答1>",
  "contact": "<回答2>",
  "instagram_account": "<回答3>",
  "age_group": "<回答4の変換値>",
  "dominant_arm": "<回答5の変換値>",
  "experience_years": "<回答6の変換値>",
  "personal_best": "<回答7>",
  "shooting_angle": "<回答8の変換値>",
  "is_slow_motion": <回答9の変換値>,
  "video_count": <回答10の変換値>,
  "main_request": "<回答11>",
  "consent_reference_analysis": true,
  "consent_not_medical": true,
  "consent_not_coaching_replacement": true,
  "consent_accuracy_depends_on_video": true,
  "consent_delivery_may_take_time": true,
  "consent_sns_requires_permission": true,
  "sns_permission_status": "<回答19の変換値>",
  "desired_plan": "beta_full_report"
}
```

### 選択肢の変換テーブル

**カテゴリ (`age_group`)**
- 高校生 → `"高校生"`
- 大学生 → `"大学生"`
- 社会人・マスターズ → `"マスターズ"`
- 指導者 → `"指導者"`
- その他 → `"一般"`

**利き腕 (`dominant_arm`)**
- 右 → `"right"`
- 左 → `"left"`
- 両方 → `"unknown"`

**撮影角度 (`shooting_angle`)**
- 真横（側面）→ `"side"`
- 斜め後方 → `"diagonal_rear"`
- 正面 → `"front"`
- その他 → `"unknown"`

**SNS掲載 (`sns_permission_status`)**
- 許可する → `"allowed"`
- 匿名なら許可 → `"anonymous"`
- 不可 → `"denied"`
- まだ決めていない → `"unknown"`
