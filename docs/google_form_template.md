# Googleフォーム 推奨テンプレート

やり投げ動画解析サービス受付用のGoogleフォーム設計ガイドです。  
CSVインポート機能（管理画面「📥 CSVインポート」タブ）と自動マッピングするために、  
以下の**推奨質問タイトル**を使用してください。

---

## 推奨フォーム構成

### セクション1: 基本情報

| 質問タイトル | 回答形式 | 必須 | 自動マッピング先 |
|---|---|---|---|
| 名前またはニックネーム | 短文テキスト | ★必須 | `name_or_nickname` |
| 連絡先（SNS・LINE名） | 短文テキスト | ★必須 | `contact` |
| 年齢区分 | プルダウン | 任意 | `age_group` |
| 性別（任意） | 短文テキスト | 任意 | `gender` |
| 競技歴 | 短文テキスト | 推奨 | `athletic_career` |
| 自己ベスト | 短文テキスト | 任意 | `personal_best` |
| 利き腕 | プルダウン | ★必須 | `dominant_arm` |
| 身長（cm） | 短文テキスト | ★必須 | `height_m` ※cmは自動でmに変換 |
| 所属区分 | プルダウン | 任意 | `affiliation_type` |

**年齢区分の選択肢:** 中学生 / 高校生 / 大学生 / 社会人 / マスターズ / その他

**利き腕の選択肢:** 右 / 左 / わからない

**所属区分の選択肢:** 中学 / 高校 / 大学 / 社会人 / マスターズ / その他

|---|---|---|---|

### セクション4: 希望プラン

| 質問タイトル | 回答形式 | 必須 | 自動マッピング先 |
|---|---|---|---|
| 希望プラン | プルダウン | ★必須 | `desired_plan` |

**希望プランの選択肢:**

| フォーム表示名 | `desired_plan` 値 |
|---|---|
| 無料プレビュー（まずお試し） | free_preview |
| ライト版（動画＋簡易コメント） | light |

| 2動画比較版（2本の動画を比較） | comparison |

### セクション5: 同意事項

以下の質問を**チェックボックス必須**で追加してください。

| 質問タイトル | 形式 | 必須 |
|---|---|---|
| 本解析は参考資料であり、絶対評価・診断結果ではないことを理解しました | チェックボックス | ★必須 |
| 本解析は医療診断・怪我の診断の代替ではないことを理解しました | チェックボックス | ★必須 |
| 本解析は専門的な競技指導の代替ではないことを理解しました | チェックボックス | ★必須 |
| 動画の画質・角度・服装等により解析精度が変わる場合があることを理解しました | チェックボックス | ★必須 |
| 解析・納品まで時間がかかる場合があることを理解しました | チェックボックス | ★必須 |
| SNSへの掲載は別途許可制であることを理解しました | チェックボックス | ★必須 |

> 同意事項のCSV列は現在自動マッピング対象外です。  
> 同意確認は管理画面「N. 受付情報」→「同意事項」セクションで手動で確認してください。

## CSVエクスポート → インポートの手順

1. Googleフォームの回答シートを Google スプレッドシートで開く
2. **ファイル → ダウンロード → CSV形式（.csv）** を選択してダウンロード
3. 管理画面の「📥 CSVインポート」タブを開く
4. CSVファイルをアップロード
5. 列名のマッピングを確認（推奨タイトルを使用していれば自動マッピング）
6. 取り込む行を選択
7. 「🚀 選択行を新規ジョブとして取り込む」ボタンをクリック

---

## 注意事項

- Googleフォームのタイムスタンプ列（`タイムスタンプ`）はスキップされます
- `身長（cm）` 列の値が 10 を超える場合、自動的に 100 で割ってメートル換算します
  - 例: `175` → `1.75`
- 利き腕のGoogleフォーム回答が「右」「left」「R」などの場合、`dominant_arm` の値は入力値そのままになります  
  管理画面で手動補正してください（`right` / `left` / `unknown`）
