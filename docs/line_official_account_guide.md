# 公式LINE運用ガイド

Javelin Video Analysis では、解析依頼の受付に **公式LINE** を活用できます。
Phase 6 では、まず **公式LINEからGoogleフォームへ誘導する運用** を推奨します。

---

## 1. 基本方針（Phase 6 での推奨運用）

**最初はシンプルに始める:**

1. 公式LINE を開設し、解析依頼受付の窓口として使う
2. Google フォームへのリンクを自動返信・リッチメニューに設置する
3. 解析完了後の納品URLは LINE のトークで送付する
4. Webhook 連携は Phase 7 以降で検討する

この運用では LINE SDK のインストールは不要で、**すぐに始められます。**

---

## 2. 自動返信メッセージ例

公式LINE の「応答メッセージ」に以下のような文章を設定してください。

### 受付案内メッセージ

```
やり投げ動画解析サービスへのお問い合わせありがとうございます。

解析依頼はこちらのフォームからお申し込みください。
→ [Googleフォームのリンクをここに入れる]

お申し込み後、確認次第ご連絡します。
納品まで数日お時間をいただく場合があります。

本解析は参考資料であり、医療診断や専門的な競技指導を代替するものではありません。
```

### 動画提出前の案内

```
動画を提出いただく前に、以下をご確認ください。

【撮影推奨条件】
・撮影角度: 側面（真横）が最も精度が出ます
・フレームレート: 60fps以上推奨（スローモーションも可）
・画質: 1080p以上推奨
・背景: シンプルな背景が望ましい
・照明: 明るい屋外や体育館照明が望ましい

【提出方法】
・Google フォームからLINEアルバムのリンク、またはGoogleドライブのURLを記載ください
・動画ファイルの直接送信も可能ですが、大容量の場合はクラウドリンクを推奨します
```

## 4. 納品URL送信文テンプレート

解析完了後に LINE で送付する文章です。

```
このたびは動画解析サービスをご利用いただきありがとうございます。

解析結果はこちらからご確認ください。

{delivery_page_url}

まずは「最初に読んでください」を開いてから、解析動画とPDFをご確認ください。

URLには有効期限があります。
期限切れの場合は再発行しますのでご連絡ください。

本解析は、動画から取得した姿勢推定データをもとにした参考資料です。
競技指導、医療判断、怪我の診断を代替するものではありません。
```

> `{delivery_page_url}` は管理画面の「P. S3納品 / 納品URL発行」セクションで生成できます。
> 生成後に「受付一覧」タブの詳細画面にも表示されます。

## 6. Webhook連携の概要（将来対応）

将来的に LINE Webhook 連携を実装する場合の概要です。

### 流れ

1. ユーザーがLINEでメッセージ送信
2. LINE Platform が Webhook URL（例: `POST /v1/line/webhook`）にイベントを送信
3. FastAPI が署名を検証して処理
4. 必要に応じて intake を自動作成

### 実装に必要なもの

- LINE Developers コンソールでチャネルを作成
- `line-bot-sdk-python` のインストール (`pip install line-bot-sdk`)
- チャネルシークレット・チャネルアクセストークンの取得
- HTTPS エンドポイント（Webhook は HTTP 不可）

### 環境変数

```
LINE_CHANNEL_SECRET=your_channel_secret
LINE_CHANNEL_ACCESS_TOKEN=your_access_token
LINE_WEBHOOK_ENABLED=true
```

> **Phase 6 では Webhook 実装は任意です。**
> まずはフォーム誘導運用で始め、依頼数が増えてから自動化を検討することを推奨します。

## 7. LINE userId の取り扱い

LINE Webhook を実装した場合、`userId` が取得できます。

**重要:**
- `userId` は LINE プラットフォームが発行するユーザー識別子です（LINE UID）
- 個人を特定するためのIDではありませんが、**個人情報と同等の扱い**が必要です
- `intake.json` の `line_user_id` フィールドに保存されますが、**ログに出力しません**
- アクセス制御を適切に行い、不要になったデータは削除してください

---

## 8. 個人情報の取り扱い注意

LINE で取得できる情報には個人情報が含まれます。

- **表示名**: LINE の表示名はニックネームが多いですが、本名の場合もあります
- **プロフィール画像**: 本人の写真の場合があります
- **userId**: LINE プラットフォームのユーザー識別子（個人情報と同等の扱い）

**管理方針:**
- LINE との会話内容のスクリーンショットを不特定多数に共有しない
- サーバーログに LINE userId・表示名を残さない
- 解析が完了したら不要な個人情報は削除する
- LINE の公式アカウントアクセス権は最小限の担当者のみに付与する
