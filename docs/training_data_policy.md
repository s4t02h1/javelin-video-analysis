# 教師データポリシー（Phase 11）

## 方針

このプロジェクトでは、将来の機械学習モデル訓練に向けて教師データ候補を管理します。

> ⚠️ **重要**: Phase 11 ではモデル訓練は行いません。
> 教師データの収集・整備のみを行います。

---

## 同意の分類

### SNS掲載許可（sns_permission）

動画・画像を SNS やウェブサイトに掲載することへの許可です。

| 値 | 意味 |
|----|------|
| `unknown` | 未確認 |
| `allowed` | 掲載許可あり |
| `anonymous` | 匿名処理すれば掲載可 |
| `denied` | 掲載不可 |

### 教師データ利用許可（consent_for_training_data）

動画・骨格データを機械学習の教師データとして利用することへの許可です。
**SNS掲載許可とは独立して管理します。**

| 値 | 意味 |
|----|------|
| `unknown` | 未確認（デフォルト） |
| `allowed` | 教師データとして利用可 |
| `anonymous_only` | 匿名化すれば利用可 |
| `denied` | 利用不可 |

> ⚠️ `sns_permission = allowed` であっても `consent_for_training_data = denied` の場合は
> 教師データとして利用しないでください。

---

## エクスポートフィルタリング

以下のルールはコードレベルで強制されます:

1. `consent_for_training_data = "denied"` → **常に除外**（例外なし）
2. `consent_for_training_data = "unknown"` → **デフォルト除外**（設定で変更可能）
3. `annotation_status = "confirmed"` でなければ除外
4. `privacy_flags` に `training_data_excluded` が付いている場合は除外
5. `source_video_path` はエクスポートに含まれない
6. 個人情報（氏名・学校名・連絡先）はエクスポートに含まれない

---

## プライバシーフラグ

| フラグ | 意味 | エクスポートへの影響 |
|-------|------|---------------------|
| `contains_face` | 顔が映っている | 警告のみ |
| `contains_school_name` | 学校名が映っている | 警告のみ |
| `contains_bib_number` | ゼッケンが映っている | 警告のみ |
| `contains_voice` | 音声が含まれる | 警告のみ |
| `contains_third_party` | 第三者が映っている | 警告のみ |
| `contains_background_info` | 背景に個人情報が映っている | 警告のみ |
| `training_data_excluded` | 教師データ利用不可 | **除外** |
| `needs_anonymization` | 匿名化が必要 | 警告のみ |
| `csv_features_only` | CSVのみ利用可（動画不可） | 警告のみ |

## 同意取得メッセージ例

以下は同意取得の参考文例です（法律専門家によるレビューが必要）:

```
【教師データ利用に関するお知らせ】

撮影いただいた動画データについて、将来の投擲競技フォーム分析AIの
改善を目的とした教師データとして使用することを検討しております。

教師データとして利用する場合でも、以下の点を守ります:
- 氏名・学校名などの個人情報は使用しません
- 骨格データ（関節座標）のみを使用します
- 動画ファイルそのものは外部に公開しません

利用を希望されない場合は、申し込み時にお知らせください。

教師データ利用: 承諾する / 承諾しない（いずれかをご選択ください）
```

## 匿名化手順

`consent_for_training_data = "anonymous_only"` の場合:
1. job_id を SHA-256 ハッシュに変換（anonymize_job_id=True の場合）
2. comparison_id / video_id を除外
3. source_video_path を除外（常に除外）
4. 個人名・学校名・連絡先を含むフィールドを除外

---

## 注意事項

- アノテーションは「完全な正解ラベル」ではなく「参考となる人間の判断記録」です
- 動画の撮影角度・画質・服装によりラベル精度が変わります
- 医療診断・怪我の診断・専門的競技指導への使用は想定していません
